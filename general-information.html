Typical neural network lifecycle is the following:

  1. Adjust neural-classifier:*learn-rate*, neural-classifier:*decay-rate* and neural-classifier:*minibatch-size* parameters if you want.
  2. Create a snakes generator which will return data for the training step in the form (CONS SAMPLE LABEL)(i.e. cons pair which contains a sample for training and a label which describes that sample).
  3. Create four functions: The first and the second translate a sample returned by the generator created in the previous step to magicl:matrix/single-float matrix with dimensions Nx1. One of them will be used by calculate and another one will be used during the training. The function used for training may make additional random transformations of the input data to extend training set. The third translates a label to magicl:matrix/single-float matrix with dimensions Mx1 and the fourth translates a matrix with dimensions Mx1to a label. Obviously the third function is used for training and the fourth is used for classifing objects with trained net.
  4. Create a neural network with neural-classifier:make-neural-network. The first parameter, layout, must be a list (N ... M), where N and M are taken from the previous step. Also is accepts created transformation functions.
  5. Train a neural network calling neural-classifier:train-epoch. An epoch is finished when supplied generator doesn't have any more data (i.e. returns snakes:generator-stop).
  6. Repeat steps 2 and 5 to train for more epochs. You can use neural-classifier:rate function to control the accuracy of your network, so you know where to stop training.
  7. After the net is trained, call neural-classifier:calculate to pass your data through the net.

If you want an example, look at mnist/mnist.lisp to see how digit recognition works.
